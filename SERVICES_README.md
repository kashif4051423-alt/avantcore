## Troubleshooting

### Issue: Images not loading
**Solution:** Check image path spelling, verify file exists in `/public/`

### Issue: Service page not found
**Solution:** Verify service ID matches (use lowercase with hyphens)

### Issue: Broken links
**Solution:** Check routing in App.tsx and Service.tsx

### Issue: Styling looks different
**Solution:** Clear browser cache, verify Tailwind CSS applied
