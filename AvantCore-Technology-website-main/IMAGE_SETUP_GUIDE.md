## Recommended Free Image Sources

1. **Unsplash** (https://unsplash.com)
   - High-quality free photos
   - No attribution required (but appreciated)
   - Search for enterprise/business/technology images

2. **Pexels** (https://www.pexels.com)
   - Free stock photos
   - No attribution required
   - Good for business/corporate images

3. **Pixabay** (https://pixabay.com)
   - Free images and videos
   - No attribution required
   - Diverse collection of business photos

## Download Instructions

1. Visit the image source website
2. Search for the keyword suggested for each image
3. Download the image in high quality (1200x800px for most)
4. Save with the exact filename specified (case-sensitive)
5. Place all images in the `/public` directory

## Image Specifications

- **Format:** JPG (for better compression) or PNG (if transparency needed)
- **Size:** 1200x800px for most images (except newsletter which is wider)
- **Optimization:** Compress images to keep file size under 200KB each
- **Style:** Professional, corporate, enterprise-looking images
- **Avoid:** Cartoonish, overly bright, or dated-looking images

## Tools for Image Compression

- TinyPNG (https://tinypng.com)
- ImageOptim (Mac)
- FileZilla (Free online compressor)
