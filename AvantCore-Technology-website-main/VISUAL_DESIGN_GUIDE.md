## IMAGE OPTIMIZATION

### File Size
- Target: <200KB per image
- Format: JPG (better compression)
- Tool: TinyPNG, ImageOptim

### Dimensions
- Standard: 1200x800px
- Newsletter: 1200x400px (wider)
- Larger screens: Up to 2400x1600px acceptable

### Compression Without Quality Loss
1. Download image at high quality
2. Resize to 1200x800px max
3. Use TinyPNG.com or similar tool
4. Compress by 60-80% to get <200KB
5. Save as JPG (quality 85-90%)
