Duration: 0.6-0.9s
Easing: easeOut
Effect: Professional appearance as content scrolls

### 2. **Hover Effects**
Used on: Cards, buttons, links, images
Duration: 0.25-0.4s
Easing: easeOut
Effect: Clear user interaction feedback

### 3. **Counter Animations**
Used on: Statistics, metrics, achievements
Duration: 0.8-1.0s

### 4. **Image Motion Effects**
Used on: Featured images, gallery items
Duration: 0.4-0.5s
Easing: easeOut
Effect: Subtle depth and dimension

### 5. **Stagger Animations**
Used on: Grids, lists, sequences
Duration: 0.08-0.2s per item
Easing: easeOut
Effect: Professional sequential reveal
